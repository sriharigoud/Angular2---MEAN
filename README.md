# Angular 2 - Mean example

App to demostrate building an angular 2 app in the MEAN stack with the Angular CLI.

## Set up.
Make sure you have angular cli installed.
```bash
$ npm install -g angular-cli
```

Make sure you have installed mongoDB. Check it's installation by typing ```mongo```

Clone the repo
```bash
$ git clone https://github.com/sriharigoud/angular2-mean.git
$ cd angular2-mean
```

Install dependencies
```bash
$ npm install
```

Run the app
```bash
$ npm run build
```
